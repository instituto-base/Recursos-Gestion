Instructivo acceso repositorios privados GitHub

Un repositorio privado es administrado por quien crea el dicho repositorio y es a través de invitaciones que otros usuarios pueden tener acceso a dichos repositorios. El administrador puede remover el derecho de colaboración en cualquier momento, y quienes sin invitados a colaborar tienen siete días para aceptar la invitación, en caso de que eso no se cumpla el administrador debe reenviar la invitación.

A continuación se utilizará el repositorio privado scripts-genomicos del Instituto Milenio BASE (instituto-base) para mostrar como aceptar las invitaciones a repositorios privados

Para poder recibir invitaciones a repositorios privados es necesario tener una cuenta en GitHub, para eso solo es necesario tener un correo electrónico activo con el cual confirmar la nueva cuenta y seguir las instrucciones de registro. Luego de crear la cuenta se debe enviar el nombre de usuario o correo electrónico asociado a el/la data manager

Luego de que la invitación se enviada, esta se puede encontrar tanto en el correo electrónico asociado a la cuenta GitHub o directamente en la sección de notificaciones

Notificación en GitHub

Correo electrónico

Si abre la solicitud desde el correo, puede ser direccionado a dos páginas distintas dependiendo si está o no la sesión iniciada de GitHub

Vista sesión iniciada (también vista desde las notificaciones)

Vista sin sesión iniciada

En este caso, solo es necesario inicial la sesión en el panel que se despliega y será redireccionado a la vista anterior

Luego de aceptar la invitación ya es posible acceder a todo el repositorio
